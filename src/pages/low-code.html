Low Code

A mesa de low-code fornece um ambiente de desenvolvimento
usado para criar software de aplicativo por meio de uma
interface gráfica de usuário.

Como funciona?

A mesa opera como um hub dinâmico, focado em capacitar os alunos a dominarem a criação de aplicativos low code. Aqui, eles aprendem a conceber soluções inovadoras, implementar sistemas eficientes usando a abordagem low code e a desenvolver programação de inteligência artificial para chatbots. Além disso, os participantes têm a oportunidade de aprimorar suas habilidades na gestão de business intelligence, proporcionando uma experiência prática e abrangente no desenvolvimento de tecnologias avançadas."

Responsabilidades

Os alunos desempenham um papel ativo e responsável. Eles aplicam prontamente os conhecimentos adquiridos, criando soluções práticas como aplicativos low code e sistemas, e programando inteligência artificial para chatbots. A colaboração e inovação são incentivadas, assim como a gestão eficiente de projetos. Participam ativamente de atividades de business intelligence, aplicando análise crítica para extrair insights valiosos. Assumindo essas responsabilidades, os alunos enriquecem sua experiência e contribuem para o sucesso coletivo da mesa.
